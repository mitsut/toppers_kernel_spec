TOPPERS Kernel Specification for asciidoc
====

TOPPERS統合仕様書をASCIIDOC形式でリライトした

## Requirement

* Ruby
* asciidoctor

## Usage

    git checkout <kernel-version>
    asciidoctor -b html book.asciidoc

## Install

    gem install asciidoctor
    gem install asciidoctor-pdf
